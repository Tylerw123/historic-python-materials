Code Fragment Example Vars

We resolved that, in order to better represent the true python ethic, we should avoid nonsensical pseudo-variable names like foo and bar, and instead use more circus-conformant names, like spam and eggs. (The possibilities are endless. Those of you that don't like spam could always just use spam, spam, spam, spam, eggs, spam, and sausage...-)
