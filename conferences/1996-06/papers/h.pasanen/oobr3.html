Supported languages

OO-Browser v02.09.11 claims support for all popular OO Languages on UNIX, giving the following list:

  1. Eiffel
  2. C++
  3. Objective-C
  4. Common Lisp and its object system (CLOS)
  5. Java
  6. Smalltalk
  7. Info documentation
