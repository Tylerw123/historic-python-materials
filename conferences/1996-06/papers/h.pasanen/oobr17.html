Appendix: Installation instructions

OO-Browser with Python support is available at

location

If you are an XEmacs user, the yet to be released XEmacs 19.14 distribution should have an OO-Browser with Python support built in. (Current release date at the time of writing is June 7) All the documentation, including the Python specific usage, is available in the info pages of XEmacs.

Older versions of XEmacs have OO-Browser installed by default but do not contain Python support. However, you will have the generic OO-Browser manuals available on-line in the info-pages, and Python usage is very similar to the general usage.
